# Quick Start Guide

## Installation

```bash
cd C:\Users\Administrator\Downloads\NLP\Tai
pip install -r requirements.txt
```

## Run Application

```bash
python app.py
```

Open browser: http://127.0.0.1:5000/

## Test Sample Data

Copy and paste content from `data/sample/news_01_ai.txt` into the textarea.

## Run Tests

```bash
python tests/test_sample.py
```

## Project Structure

- `core/` - Core NLP modules
- `templates/` - HTML templates
- `static/uploads/` - Generated visualizations
- `data/sample/` - Sample texts
- `evaluation/` - Evaluation results
